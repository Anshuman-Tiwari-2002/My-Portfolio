## Local Development

Clone the repository:

```bash
git clone https://github.com/Anshuman-Tiwari-2002/My-Portfolio.git
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
